Yellow Taxi Trips:
    1.Vendor Id not null and integer type
    2.6>passenger_count>0 and not null
    3.trip_distance>0 and not null
    4.RateCodeId!=null if payment_type!=0 and 1 through 6 + 99
    5.store and fwd flag!=null if payment_type!=0,values=Y,N
    6.PULocationId and DropOff LocationId != null and integer type, PU or DO = JFK when rate code =2 PU or DO=Newark when rate code =3
    7.pickup_datetime should be less than dropoff_datetime
    8.payment_type!= null and integer type
    9.fare_amount !=null and fare_amount>3 when RateCodeId=1,
    fare_amount>= 52 when RateCodeId=2,fare_amount>20 when RateCodeId=3
    extra>=0 and !=null
    10.mta_tax=0 or 0.5 and != null (DO not ouside nyc)
    11.tip_amount>0 when payment_type=1 else 0
    12.tolls_amount>=0 and != null
    13.total_amount must be approx equal to sum of all fares and charges and !=null
    14.improvement_surcharge>=0.3 and !=null (always levied)
    15.congestion_surcharge=2.5 or 0 and != null (only for PU and DO in manhattan zone)
    16.airport fee = 0 or 1.75 (if pickup id is JFK or newark) and != null
